About this web application

This project is a Book Management System, using Spring Boot 2 and Angular 9.
It's a end-to-end web application with a connectivity to Oracle database.
Back end application is a RESTful web services using Spring Boot 2.
It performs all the CRUD operations:

  • create a book
  • read a single book or a list of books
  • delete a book
  • update a book
Back End Technologies used
  • Java 8
  • Spring Boot 2.3.3
  • Spring Data JPA
  • Maven 4
  • Hibernate 5
  • Tomcat 9
  • Oracle 11g
  • SLF4J
Front End Technologies used
  • Angular 9
  • Angular CLI
  • Bootstrap 4
End Points
  • GET /books
  • GET /books/{bookid}
  • POST /books
  • PUT /books/{bookid}
  • PATCH /books/{bookid}
  • DELETE /books/{bookid}
